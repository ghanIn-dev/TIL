# int vs integer

## int

* 자료형

* 산술 연산 가능

* null 초기화 불가능 \/ 0으로 초기화


## integer

* 클래스

* unboxing을 하지 않으면 산술 연산이 불가능하지만, null 값을 처리가능

* null값 처리가 용이해서 sql 연동시 사용함, 직접적인 산술연산은 불가능
